# Mansoldev Framework CSS

A modern CSS framework powered by Sass and PostCSS.

## How to use:

For testing purpose:

```
pnpm run dev
```

Build ready to use:

```
pnpm run build
```

## Project Architecture:

The project will consist:

- The CSS framework itself in src/ folder that builds in dist/ folder
- An astro project as interactive documentation in website/ forder

```
framework/
│
├── src/                     # source code
│   ├── scss/
│   ├── utilities/
│   └── components/
│
├── dist/                    # Compiled output
│   ├── main.css
│   └── main.min.css
│
├── website/                 # Astro proyect: documentation source code
│   ├── src/
│   │   ├── pages/
│   │   ├── layouts/
│   │   ├── components/
│   │   └── styles/
│   │       └── website.scss
│   │
│   ├── public/
│   └── astro.config.mjs
│
├── docs/                    # Output generated by Astro
│
├── package.json
├── postcss.config.js
└── README.md
```
